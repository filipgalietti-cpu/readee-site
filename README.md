# FillySteeze Accounting Academy

FillySteeze Accounting Academy is a single-page learning experience designed to help beginners build accounting fundamentals through interactive demos, flashcards, quizzes, and practice tools.

## Features

- **Interactive journal entry demo** – Drag and drop accounts into debit and credit columns, then check your work for instant feedback.
- **Modular learning path** – Expandable sections outline the recommended sequence of accounting topics.
- **Tap-to-flip flashcards** – Reinforce key terms and definitions with accessible flashcards.
- **Dynamic quiz** – Randomized multiple-choice questions with explanations to confirm understanding.
- **Financial calculators** – Break-even and current ratio calculators for hands-on practice with managerial accounting concepts.

## Getting Started

1. Open `index.html` in your browser.
2. Explore the interactive widgets and calculators across each section.

All styling lives in `assets/css/styles.css` and the behavior is implemented in `assets/js/app.js`.
